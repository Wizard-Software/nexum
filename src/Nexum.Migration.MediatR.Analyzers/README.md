# Nexum.Migration.MediatR.Analyzers

Roslyn analyzer providing migration hints from MediatR to [Nexum](https://github.com/Wizard-Software/nexum).

This package is bundled with `Nexum.Migration.MediatR` — you don't need to install it separately.

## Documentation

Full documentation: **[nexum.wizardsoftware.pl](https://nexum.wizardsoftware.pl)**

See [Migration from MediatR](https://nexum.wizardsoftware.pl/articles/migration-from-mediatr.html) for the full step-by-step migration guide.
