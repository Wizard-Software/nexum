# Nexum.Streaming

Streaming notifications, SignalR integration, and SSE endpoints for [Nexum](https://github.com/Wizard-Software/nexum).

## Installation

```bash
dotnet add package Nexum.Streaming
```

## Usage

Push domain events to clients in real-time via SignalR hubs or Server-Sent Events (SSE) endpoints, with built-in backpressure and filtering.

## Documentation

Full documentation: **[nexum.wizardsoftware.pl](https://nexum.wizardsoftware.pl)**

See [Stream Queries](https://nexum.wizardsoftware.pl/articles/streams.html) for detailed usage.
