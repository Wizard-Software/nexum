# Nexum.Streaming

Streaming notifications, SignalR integration, and SSE endpoints for [Nexum](https://github.com/asawicki/nexum).

## Installation

```bash
dotnet add package Nexum.Streaming
```

## Usage

Push domain events to clients in real-time via SignalR hubs or Server-Sent Events (SSE) endpoints, with built-in backpressure and filtering.
