# Nexum.Migration.MediatR

MediatR migration adapters for [Nexum](https://github.com/Wizard-Software/nexum). Enables gradual handler-by-handler migration.

## Installation

```bash
dotnet add package Nexum.Migration.MediatR
```

## Usage

Run MediatR and Nexum side-by-side — migrate handlers one at a time without a big-bang rewrite. The package includes a bundled Roslyn analyzer that provides migration hints and warnings.

## Documentation

Full documentation: **[nexum.wizardsoftware.pl](https://nexum.wizardsoftware.pl)**

See [Migration from MediatR](https://nexum.wizardsoftware.pl/articles/migration-from-mediatr.html) for the full step-by-step migration guide.
